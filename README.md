# Project Phoenix — Linux Foundations

This repository documents the first Linux learning module of Project Phoenix.

## Objective

Build a strong practical foundation in Linux system administration using a dedicated Ubuntu workstation and a real HomeLab environment.

## Learning Approach

- Learn by building.
- Understand before automating.
- Document every meaningful change.
- Practice troubleshooting and recovery.
- Keep configurations reproducible.
- Treat security and backups as core requirements.

## Environment

- Ubuntu 26.04 LTS Desktop
- Fish shell
- Git
- Lenovo ThinkPad T14 Gen 2
- Project Phoenix HomeLab

## Module Topics

- Linux system architecture
- Filesystem navigation
- Users and groups
- File permissions
- Processes and services
- Package management
- Storage and memory
- Logging and troubleshooting
- Basic networking
- Security, backup, and recovery

## Day 1 — Environment Setup

- Installed Git.
- Configured Fish as the default login shell.
- Installed Visual Studio Code from the official `.deb` package.
